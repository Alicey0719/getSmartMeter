# getSmartMeter

## init
1. create conf.ini
```
[settings]
broute_id = 
broute_pw = 
```

2. pip install
```
python3 -m pip install -r requirements.txt
```

2. run get_smart_meter_para.py
```
python3 get_smart_meter_para.py
```
conf.iniが増える

## run
1. run get script
```
python3 show_current_watt.py
```
